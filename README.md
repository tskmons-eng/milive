# milive

Merged Chrome extension for MLive, Carsensor, MotorGate, Arai AA, and JU Navi workflow tools.

## Included tools

- MLive image-cell linkifier and image downloads
- Carsensor/MotorGate copy, import, search, defaults, and standard image upload
- Carsensor inquiry reply templates and reply-page helpers
- Arai AA new-tab override and image downloads
- JU Navi list automation, navigation shortcut, and image downloads
